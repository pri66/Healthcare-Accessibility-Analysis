## Project Overview

This project evaluates healthcare accessibility in Jabalpur District, Madhya Pradesh using GIS. Existing hospital service coverage was analysed using 5 km buffer analysis, accessibility gaps were identified by locating settlements outside the service areas, and suitable locations for new hospitals were proposed using spatial decision-making.

---

## Objectives

- Assess existing healthcare accessibility.
- Identify underserved settlements.
- Prioritize tehsils requiring healthcare intervention.
- Recommend suitable hospital locations.
- Develop a GIS dashboard for decision support.

---

## GIS Tools & Techniques

- QGIS
- Buffer Analysis
- Overlay Analysis
- Extract by Location
- Count Points in Polygon
- Spatial Statistics
- Cartographic Layout Design

---

## Key Findings

- Hospital Service Distance: **5 km**
- Uncovered Settlements: **498**
- Highest Priority Tehsil: **Majholi (131)**
- Second Priority: **Shahpura (90)**
- Third Priority: **Sihora (77)**
- Proposed Hospital Sites: **3**
